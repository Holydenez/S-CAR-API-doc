get /v1/spareparts
(v1SparepartsGet)
Returns content of spareparts objects (array) and info about pages. If **articleNumber** is specified, returns only the objects that contain those param. ***None of params are required.***

Query parameters

page (optional)
Query Parameter — The number of page. List of objects is divided into pages according to a size param. ***Default = 0*** format: int
size (optional)
Query Parameter — The size of elements to be in this array of objects. ***Default = 30*** format: int
articleNumber (optional)
Query Parameter — Number of sparepart

Return type

array[array[Object]]

Example data

Content-Type: application/json
[ [ "{}" ] ]
